# SmartARV_Skill_Chek_Application

SmartARV Skill Check is a Java-based application developed to assess students' skills through an interactive examination system. The application allows users to log in, select exam categories (Aptitude, English, Reasoning, and Mixed), attempt questions within a time limit, and receive automatic evaluation with results and feedback.

**Technologies Used:** Java, Java Swing, Object-Oriented Programming (OOP).

This project demonstrates the implementation of GUI-based applications, authentication, timed exams, and result evaluation using Java
